### Задача 1: Обеспечить разработку
Предлагаю использовать GitLab\
Он соответствует всем заданным требованиям, т.к. изначально создавался как приложение для управления кодом и CI/CD\
Для обеспечения большей безопасности возможно использование GitLab совместно с HashiCorp Vault
### Задача 2: Логи
Предлагаю использовать ELK\
Готовый стек для сборка, хранения и визуализации логов, который соответствует всем заданным требованиям\
Ну а ещё я знаком с Kibana как пользователь и она мне более чем нравится :)
### Задача 3: Мониторинг
Предлагаю использовать Prometheus + Grafana\
В совокупности они так же соответствуют всем заданным требованиям\
Prometheus - достаточно простой и надёжный инструмент для сбора метрик\
Grafana - без проблем взаимодействет с Prometheus, есть готовые дашборды для метрик\
Prometheus + Grafana = одно из лучших решений для мониторинга (возможно даже является стандартом для большинства случаев)
